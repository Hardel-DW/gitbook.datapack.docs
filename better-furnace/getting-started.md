# Getting Started

## <mark style="color:orange;">What is BetterFurnace?</mark>

**BetterFurnace** is a datapack developed by Hardel that introduces **seven new types of furnaces** with varying cooking speeds. Each furnace gets progressively faster and harder to craft, fitting perfectly with your Minecraft survival experience.

The best part? **No Resource Pack required!**

## <mark style="color:orange;">Furnace Types & Speeds</mark>

| Furnace | Cooking Speed |
| --- | --- |
| **Iron Furnace** | 4 Seconds |
| **Copper Furnace** | 3 Seconds |
| **Golden Furnace** | 2 Seconds |
| **Diamond Furnace** | 1 Second |
| **Emerald Furnace** | 0.5 Seconds |
| **Netherite Furnace** | 4 Items per second |
| **Extreme Furnace** | 20 Items per second |

---

## <mark style="color:orange;">Features</mark>

- **7 unique furnace tiers** with increasing cooking speeds
- **Progressive difficulty** — harder furnaces require better materials
- **No Resource Pack needed** — works with vanilla textures
- **Survival-friendly** — balanced crafting recipes
- **Fully compatible** with standard furnace mechanics
