# Yggdrasil Alternative

## <mark style="color:orange;">Overview</mark>

**Yggdrasil Alternative** is a completely new variant of the original Yggdrasil pack. This version removes all custom dimensions and replaces problematic mechanics to provide a lighter, more compatible experience perfect for multiplayer servers.

The Alternative version delivers the core structures and storyline without dimension-related compatibility issues, making it fully compatible with **Paper, Bukkit, Purpur**, and other lightweight server implementations.

---

## <mark style="color:orange;">Key Differences</mark>

### No Custom Dimensions

The most significant change: **all custom dimensions have been removed**. This eliminates compatibility issues with various server software and reduces performance overhead.

**Runic Dimension Content Relocated to Nether:**
- **Runic Labyrinth** — Now accessible from the Nether roof
- **Alfheim Nether Tree** — Provides access to Nether roof
- **Cuboid** — Relocated to the Nether roof
- **Runic Island** — Relocated to the Nether roof
- **Dokkalfar Ruins** — Relocated to Nether roof

### Simplified Reward System

**Copper Chests Replace Vaults:**
- Trial vaults and ominous trial vaults have been replaced with copper chests
- Eliminates vault-related bugs
- Improves compatibility across server implementations
- Reduces technical overhead

**Removed Items and Systems:**
- **Divine Keys** — No longer needed
- **Vault Keys** — Removed along with vaults
- **Divine Villagers** — The 5 key-exchange villagers removed
- **Reva (Familiar)** — The shoulder-perching companion removed

**Modified Content:**
- **Asflors Quest** — Updated to work with copper chests
- **The Runic Tree** — Completely removed, replaced by Alfheim Nether Tree

## <mark style="color:orange;">Why Choose Alternative?</mark>

### Full Server Compatibility

✅ **Paper** — Full support  
✅ **Bukkit** — Full support  
✅ **Purpur** — Full support  
✅ **Spigot** — Full support  
✅ **Other lightweight server implementations** — Generally compatible  
✅ **Huge Datapacks/Modpacks** — Better compatibility  

### No Dimension Issues

Custom dimensions are the primary source of compatibility problems. Alternative eliminates:
- Dimension loading errors
- Cross-server teleportation issues
- Performance degradation from dimension management
- Incompatibility with certain server plugins

## <mark style="color:orange;">Installation</mark>

**Requirements:**
- Minecraft 1.21.11
- Works with vanilla servers and all major server implementations
- Compatible with NeoEnchant 5.13.0

**Important Notes:**
- **Cannot coexist** with regular Yggdrasil — choose one version
- Don't remove regular Yggdrasil if you have it, it's a dimension-based version
