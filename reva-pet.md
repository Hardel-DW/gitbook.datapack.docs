# Reva (Pet)

## <mark style="color:orange;">Obtaining the Helheim Pet (Reva):</mark>

First, ensure you have located and entered Helheim. Please refer to the main Helheim page for details on finding and accessing the structure. Once inside, follow the steps below:

To acquire the unique pet companion from Helheim, you must follow its specific quest. This involves:

{% stepper %}
{% step %}
### Quest Books

Thoroughly exploring Helheim and locating all the lore books detailing its tragic history.
{% endstep %}

{% step %}
### Locate the room

Ascending to the structure's final floor, it's the highest room in a tower.
{% endstep %}

{% step %}
### Place books

Placing each of the read lore books onto the designated lecterns found ther.

Completing this task will reveal a hidden chest. Inside, you'll find information leading you to the location of Reva, the Helheim pet.

{% hint style="info" %}
If you wish to bypass the lore-gathering aspect, simply placing any written book onto each of the lecterns on the final floor will also trigger the chest's appearance.
{% endhint %}
{% endstep %}

{% step %}
### Find Reva

Reva is in the first room of the tower, under the slave tree. Precisely under this block, which differs from the others

{% endstep %}
{% endstepper %}
