# Special: Auto Smelt

## <mark style="color:orange;">Introduction</mark>

This enchantment is a bit special and cannot easily be modified for balancing purposes.

### <mark style="color:red;">Modding</mark>

it works with LootTables files, and modifies the loots tables of 80 blocks. So it's not easy to add blocks from modding.

### <mark style="color:red;">Remove Effect</mark>

it works with LootTables files, and modifies the loots tables of 80 blocks. So it's not easy to add blocks from modding. `data/minecraft/loot_table/blocks`
